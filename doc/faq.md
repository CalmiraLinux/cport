# Часто Задаваемые Вопросы

## 1. Что такое cport и cport API?

> 1. cport API - это API менеджера системы портов cport.

> 2. cport - программа для управления программным обеспечением, находящимся в
>    системе портов. Грубо говоря, это софт для управления этой системой портов,
>    так как помимо управления ПО cport в будущем будет способен обновлять
>    систему портов до актуальной версии, проверять её на корректность и
>    выполнять прочие манипуляции с ней.

## 2. Какие действия умеет на данный момент cport?

> На данный момент cport находится в разработке. Работа над cport API закончена
> лишь на 20%.

## 3. cport - это свободное ПО?

> Да, cport является свободным программным обеспечением. Он распространяется под
> лицензией GNU GPLv3. В случае, если вы нашли какое-либо нарушение лицензии, то
> создайте issue об этом на
> [GitHub](https://github.com/CalmiraLinux/cport/issues/new).
